Recap of behavior
  • Reasoning chip — always visible when the model exposes effort levels. Click opens a small menu to pick low/medium/high/xhigh/max/ultrathink.
  • Fast Mode chip — rendered only when the model supports it (Opus 4.6 / 4.5). Click toggles in place; yellow when on, dim outline when off.
  • Context window chip — always visible when the model exposes it. Click opens a small menu (200k / 1M).
  • Thinking chip (Haiku 4.5) — same brain icon as Reasoning, cyan tint when on. Click toggles.
  • Ultrathink — special: gradient pink/violet with sparkle icon, never represented by dots. Selecting it via the Reasoning menu can also prefix the prompt with Ultrathink: (existing behavior preserved).
  • Trigger label string (Fast · High · 200k) is gone. Chips replace it; the menu now opens by clicking each chip directly.
